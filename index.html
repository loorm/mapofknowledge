Solving one-step equations
The core idea
An equation is a statement that two expressions are equal. To solve it, you need to find the value of the unknown variable that makes both sides balance — like a set of scales.
x + a = b  ⟹  x = b − a
Whatever you do to one side, do to the other
Worked example — Solve 2x + 3 = 11
1
Subtract 3 from both sides:  2x = 8
2
Divide both sides by 2:  x = 4
3
Check: 2(4) + 3 = 11 ✓
Key principle
Use inverse operations: if the equation adds, you subtract; if it multiplies, you divide. Always perform the same operation on both sides to keep the equation balanced.
